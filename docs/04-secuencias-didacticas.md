# 04 — Secuencias Didácticas

## 1. Introducción

Este documento describe las **secuencias de juego** de cada actividad desde la perspectiva del aprendizaje. Cada secuencia detalla:

- **Qué hace el niño** (interacción)
- **Qué aprende** (proceso cognitivo)
- **Cómo lo apoyamos** (andamiaje)
- **Qué esperamos observar** (evaluación formativa)

---

## 2. Secuencia Didáctica: Canastas Numéricas

### 2.1 Estructura General

```
Inicio → Elección de nivel → Exploración → Práctica → Verificación → Feedback → Cierre
```

### 2.2 Fase 1: Activación (Menú Principal)

| Elemento | Descripción |
|----------|-------------|
| **Acción del niño** | Escoge nivel (Fácil, Medio, Difícil) |
| **Proceso cognitivo** | Toma de decisiones, autoevaluación de capacidad |
| **Andamiaje** | Los niveles están etiquetados con rangos numéricos (1-3, 1-5, 1-10) y colores (verde-amarillo-rojo) |
| **Lenguaje sugerido para el adulto** | *"¿Quieres intentar con números pequeños o te sientes listo para números más grandes?"* |

🧑‍🏫 **Sugerencia para el aula**: Antes de dejar al niño escoger, el docente puede preguntar "¿Hasta qué número sabes contar?" y orientar la elección.

---

### 2.3 Fase 2: Exploración (Disposición del Material)

| Elemento | Descripción |
|----------|-------------|
| **Acción del niño** | Observa las canastas con sus números, ve las manzanas disponibles |
| **Proceso cognitivo** | Reconocimiento de numerales, estimación inicial de la tarea |
| **Andamiaje** | Las canastas están espaciadas, los números son grandes y contrastantes |
| **Indicador de logro esperado** | El niño puede señalar y decir "este número es..." (reconocimiento) |

🧑‍🏫 **Sugerencia para el aula**: Pedir al niño que **toque cada canasta** y escuche el número (feedback auditivo) antes de empezar.

---

### 2.4 Fase 3: Ejecución (Colocación de Manzanas)

#### Subfase 3a: Arrastre inicial

| Elemento | Descripción |
|----------|-------------|
| **Acción del niño** | Toma una manzana del área de origen y la arrastra a una canasta |
| **Proceso cognitivo** | Subitización (cantidades pequeñas) o conteo uno a uno (cantidades mayores) |
| **Estrategia esperada** | El niño puede: (a) tomar manzanas una por una contando en voz alta, (b) tomar un grupo y contarlas dentro de la canasta |
| **Error común** | Llenar todas las canastas sin contar → se verá al verificar |
| **Andamiaje** | Al arrastrar, la canasta se ilumina (drag-over feedback) |

#### Subfase 3b: Ajuste

| Elemento | Descripción |
|----------|-------------|
| **Acción del niño** | Quita o añade manzanas según su propia revisión |
| **Proceso cognitivo** | Comparación (cantidad actual vs. número objetivo), corrección de errores |
| **Indicador de logro** | El niño puede explicar: *"Aquí puse 3 pero el número dice 4, necesito una más"* |

🧑‍🏠 **Sugerencia para el aula**: El docente puede preguntar *"¿Cómo sabes que ya está bien?"* para fomentar la metacognición.

---

### 2.5 Fase 4: Verificación (Evaluación Formativa)

| Elemento | Descripción |
|----------|-------------|
| **Acción del niño** | Presiona "¡Verificar canastas!" 🔍 |
| **Proceso cognitivo** | Juicio, autoevaluación, comparación |
| **Posibles resultados** | |
| ✅ **Todo correcto** | Canastas se marcan en verde, sonido de éxito, luego pantalla de victoria con confeti |
| ❌ **Alguna incorrecta** | Canastas incorrectas se marcan en rojo, invitación a revisar |
| **Andamiaje** | El color (verde/rojo) da feedback inmediato. Adicionalmente se podría añadir feedback textual/cuantitativo |

🧑‍🏫 **Sugerencia para el aula**: Si el niño se equivoca, el docente puede guiar: *"Veamos... la canasta roja dice 3. ¿Cuántas manzanas tienes ahí? Cuéntalas."*

---

### 2.6 Fase 5: Celebración (Cierre)

| Elemento | Descripción |
|----------|-------------|
| **Acción del niño** | Ve el confeti, escucha la celebración, presiona "Jugar de Nuevo" |
| **Proceso cognitivo** | Satisfacción por logro, consolidación de la experiencia positiva |
| **Refuerzo** | Mensaje de "¡Excelente Trabajo!" + opción de repetir o subir de nivel |

## 3. Secuencia Didáctica: Aventura Numérica Backyardigans

### 3.1 Estructura General

```
Inicio → Personaje presenta problema → Niño responde → Feedback → Siguiente nivel → Cierre
```

### 3.2 Fase 1: Motivación (Pantalla de Inicio)

| Elemento | Descripción |
|----------|-------------|
| **Acción del niño** | Ve la pantalla con personajes, presiona "Comenzar" |
| **Proceso cognitivo** | Activación de conocimientos previos (¿conoce a los Backyardigans?) |
| **Andamiaje** | Canción inicial, personaje animado (Pablo) salta |

---

### 3.3 Fase 2: Problema Contextualizado (Cada Nivel)

#### Nivel 1 — Pablo (Conteo)

| Elemento | Descripción |
|----------|-------------|
| **Contexto** | Pablo tiene galletas 🍪 y necesita saber cuántas hay |
| **Tarea** | Contar 3 galletas y seleccionar el número 3 |
| **Habilidad** | Conteo uno a uno, cardinalidad |
| **Error típico** | Contar rápido y no asignar correctamente (decir 2 en lugar de 3) |

#### Nivel 2 — Tyrone (Identificación de Numeral)

| Elemento | Descripción |
|----------|-------------|
| **Contexto** | Tyrone busca una casa con el número 2 |
| **Tarea** | Identificar el numeral 2 entre opciones |
| **Habilidad** | Reconocimiento de numerales |
| **Error típico** | Confundir 2 con 5 (espejo) o con otros números |

#### Nivel 3 — Tasha (Correspondencia Cantidad-Capacidad)

| Elemento | Descripción |
|----------|-------------|
| **Contexto** | Tasha tiene 4 flores y necesita una caja que las guarde |
| **Tarea** | Elegir la caja con capacidad 4 |
| **Habilidad** | Correspondencia, comprensión de "suficiente" |
| **Error típico** | Elegir por tamaño visual de la caja en lugar del número |

## 4. Progresión Recomendada

Para un niño que juega por primera vez, se recomienda esta secuencia:

### Sesión 1 (10-15 minutos)

```
1. Aventura Backyardigans (completo)
   → Familiarización con el formato de juego
   → Conexión emocional con personajes
2. Canastas Numéricas - Nivel Fácil
   → Práctica de correspondencia con cantidades pequeñas
   → Introducción a la mecánica de arrastre
```

### Sesión 2 (10-15 minutos)

```
1. Canastas Numéricas - Nivel Medio
   → Ampliación del rango numérico (1-5)
   → Mayor exigencia de conteo sistemático
2. Repaso de Aventura Backyardigans
   → Consolidación
```

### Sesión 3+ (según progreso)

```
1. Canastas Numéricas - Nivel Difícil
   → Cantidades hasta 10
   → Estrategias de conteo avanzadas
2. Juego libre (el niño elige nivel)
```

## 5. Estrategias de Andamiaje (Propuestas de Mejora)

Actualmente, el juego tiene andamiaje mínimo (niveles de dificultad). Propuestas para implementar:

| Estrategia | Descripción | ¿Cuándo activarla? |
|------------|-------------|-------------------|
| **Pista visual** | Mostrar la cantidad de manzanas que faltan (ej: "Faltan 2 🍎") | Después de 2 intentos fallidos de verificación |
| **Conteo guiado** | El juego cuenta en voz alta mientras se arrastra cada manzana | Después de 3 manzanas mal colocadas en una canasta |
| **Demostración** | Una mano animada coloca una manzana correctamente como ejemplo | Primera vez que se juega un nivel |
| **Reducción de opciones** | En Aventura Backyardigans, reducir a 2 opciones si falla | Después de 1 intento fallido |
| **Autoverificación incremental** | El botón "Verificar" se habilita solo cuando se ha colocado al menos 1 manzana en cada canasta | Siempre |

## 6. Transferencia: Del Juego a la Vida Real

Para maximizar el aprendizaje, se sugieren actividades desconectadas:

| Concepto del juego | Actividad fuera de la pantalla |
|--------------------|-------------------------------|
| Correspondencia número-cantidad | Colocar cubiertos en la mesa: "Necesitamos 4 platos, ¿puedes contar cuántos has puesto?" |
| Conteo uno a uno | Contar los escalones al subir, los pasos hasta la puerta |
| Cardinalidad | "¿Cuántos lápices hay en el estuche? Cuéntalos. ¿Cuál fue el último número?" |
| Comparación | "¿Hay más manzanas rojas o verdes en la frutera?" |
